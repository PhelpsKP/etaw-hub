# PT Hub Lite

Lightweight custom training hub for Elite Training & Wellness.

---

## Frontend (React + Vite + TypeScript)

Commands:

cd frontend  
npm install  
npm run dev  

Runs locally at:  
http://localhost:5173

---

## Backend (Cloudflare Worker)

Commands:

cd backend  
npm install  
npm run dev  

Runs locally at:  
http://127.0.0.1:8787

---

## Tech Stack

Frontend:
- React
- Vite
- TypeScript

Backend:
- Cloudflare Workers
- TypeScript

Tooling:
- npm
- Wrangler
